Event Booking API
A RESTful API built with Node.js, Express.js, and MongoDB that allows users to book events while enabling admins and event creators to manage events.

Features include JWT authentication, role-based access control, data validation, and real-time capacity management
 Features
✅ User registration & authentication (JWT)
✅ Role-based access: Admins & normal users
✅ Create, update, delete events (with ownership check)
✅ Book & cancel event bookings (with capacity management)
✅ Data validation using express-validator
✅ Error handling middleware & clean code structure

Tech Stack
Node.js + Express.js – Server & routing

MongoDB + Mongoose – Database & ODM

JWT (jsonwebtoken) – Authentication

express-validator – Request validation

bcryptjs – Password hashing
